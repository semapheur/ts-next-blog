# Units

| Quantity                | SI name      | SI symbol                       | SI base unit                                                                                         | Gaussian                | Conversion factor           |
|-------------------------|--------------|---------------------------------|------------------------------------------------------------------------------------------------------|-------------------------|-----------------------------|
| Time                    | second       | s                               |                                                                                                      | second                  | $1$                         |
| Length                  | metre        | m                               |                                                                                                      | centimetre              | $10^2$                      |
| Mass                    | kilogram     | kg                              |                                                                                                      | gram                    | $10^3$                      |
| Electric current        | ampere       | A                               |                                                                                                      | esu/second (statampere) | $3\cdot 10^9$               |
| Temperature             | kelvin       | K                               |                                                                                                      |                         |                             |
| Frequency               | hertz        | Hz                              | $\frac{1}{\mathrm{s}}$                                                                               |                         |                             |
| Force                   | newton       | N                               | $\frac{\mathrm{kg}\cdot \mathrm{m}}{\mathrm{s}^2}$                                                   | dyne                    | $10^5$                      |
| Energy                  | joule        | J                               | $\mathrm{N}\cdot\mathrm{m} = \frac{\mathrm{kg}\cdot \mathrm{m}^2}{\mathrm{s}^2}$                     | erg                     | $10^7$                      |
| Power                   | watt         | W                               | $\frac{\mathrm{J}}{\mathrm{s}} = \frac{\mathrm{kg}\cdot \mathrm{m}^2}{\mathrm{s}^3}$                 | erg/second              | $10^7$                      |
| Electric charge         | coloumb      | C                               | $\mathrm{A}\cdot\mathrm{s}$                                                                          | esu (statcoloumb)       | $3\cdot 10^9$               |
| Electric potental       | volt         | V                               | $\frac{\mathrm{J}}{\mathrm{C}} = \frac{\mathrm{kg}\cdot \mathrm{m}^2}{\mathrm{A}\cdot\mathrm{s}^3}$  | statvolt                | $300^{-1}$                  |
| Electric resistance     | ohm          | $\Omega$                        | $\frac{\mathrm{kg}\cdot \mathrm{m}^2}{\mathrm{A}^2\cdot\mathrm{s}^3}$                                | second/centimeter       | $\frac{1}{9}\cdot 10^{-11}$ |
| Capacitance             | farad        | F                               | $\frac{\mathrm{C}}{\mathrm{V}} = \frac{\mathrm{A}^2\cdot\mathrm{s}^4}{\mathrm{kg}\cdot\mathrm{m}^2}$ | centimeter              | $9\cdot 10^{11}$            |
| Inductance              | henry        | H                               | $\Omega\cdot\mathrm{s} = \frac{\mathrm{kg}\cdot\mathrm{m^2}}{\mathrm{A}^2\cdot\mathrm{s}^2}$         | second^2/centimeter     | $\frac{1}{9}\cdot 10^{-11}$ |
| Magnetic flux density   | tesla        | T                               | $\frac{\mathrm{kg}}{\mathrm{A}\cdot\mathrm{s}^2}$                                                    | gauss                   | $10^4$                      |
| Magnetic flux           | weber        | Wb                              | $\frac{\mathrm{kg}\cdot\mathrm{m}^2}{\mathrm{A}\cdot\mathrm{s}^2}$                                   | maxwell                 | $10^8$                      |
| Magnetic field strength | ampere/meter | $\frac{\mathrm{A}}{\mathrm{m}}$ |                                                                                                      | oersted                 | $4\pi \cdot 10^{-3}$        |
