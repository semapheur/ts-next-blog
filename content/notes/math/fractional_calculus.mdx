# Riemann-Liouville formulation

Recall that the repeated integral of a continuous real function $f$ is given by the Cauchy formula

$$
  I_{a^+}^n f(x) = \frac{1}{(n - 1)!}\int_a^x (x - t)^{n-1} f(t)\,\mathrm{d}t,\quad -\infty < a \leq x < b \leq \infty, n\in\mathbb{N}
$$

The Riemann-Liouville integral extends this formula from positive integer values $n\in\mathbb{N}_+$ to positive real values $\alpha$ by replacing $(n - 1)! = \Gamma(n)$. Using the Gamma function, the  Riemann-Liouville fractional integral of order $\alpha$ is defined as

$$
  I^n f(x) = \frac{1}{\Gamma (n)}\int_a^x (x - t)^{n-1} f(t) \mathrm{d}t
$$

Properties
1. $I^{a + b} f = I^a I^b f$
2. $\frac{\mathrm{d}}{\mathrm{d}x}I^{a+1} = I^a$
