# Test functions

<MathBox title='Partial derivative' boxType='definition'>
Let $\alpha = (\alpha_1,\dots,\alpha_n) \in\N_0^n$ be a multiindex of order $n$ with length $|\alpha| = \sum_{i=1}^n \alpha_i$. For $\alpha, \beta \in \N_0^n$ and $\mathbf{x}\in\R^n$, we define the following notations

- $\alpha \leq \beta$ if $\alpha_i \leq \beta_i$ for $i=1,\dots,n$
- $\binom{\beta}{\alpha} = \frac{\beta!}{\alpha! (\beta - \alpha)!}$, where $\alpha! = \prod_{i=1}^n \alpha_i!$
- $\mathbf{x}^\alpha = \prod_{i=1}^n x_i^{\alpha_i}$

We denote by $\D^\alpha f$ the partial derivative of order $|\alpha|$ of a function $f:\Omega\subset\R^n \to\Gamma$, defined as

$$
  \D^\alpha f = \frac{\partial^{|\alpha|}}{\partial x_1^{\alpha_1}\cdots\partial x_n^{\alpha_n}} f
$$

where $\D^\alpha = \D_1^{\alpha_1} \cdot D_n^{\alpha_n}$ and $\D_j = \partial/\partial x_j$ for $j = 1,\dots,n$. If the function $f$ has continuous partial derivative up to the order $|\alpha + \beta|$ inclusively, then

$$
  \D^{\alpha + \beta} = \D^\alpha (\D^\beta f) = \D^\beta (\D^\alpha f)
$$

We denote by $C^m (\Omega)$ the of functions $f:\Omega\subset\R^n \to\Gamma$ with continuous derivative of order $m$, that is, $\D^\alpha f$ is continuous on $\Omega$ for every $\alpha$ with $|\alpha| \leq m$. When $m = 0$, we have the set $C^0 (\Omega)$ of continuous functions on $\Omega$. The set of functions on $\Omega$ with continuous derivatives of all orders is denoted

$$
  C^\infty (\Omega) := \bigcap_{k\in\N} C^k (\Omega)
$$

and is called the space of smooth functions. Clearly, we have $C^\infty (\Omega) \subset C^m (\Omega) = C^0 (\Omega)$ for all $m > 0$.
</MathBox>

<MathBox title='Support' boxType='definition'>
The support of a function $f:\R^n \to\Gamma$ is the closure of the set of points where $f$ is not zero, i.e.

$$
  \operatorname{supp}(f) := \overline{\set{\mathbf{x}\in\R^n | f(\mathbf{x}) \neq 0}}
$$

If $\mathbf{x}_0 \in\operatorname{supp}(f)$, then for all $r$-balls $\mathcal{B}_r (\mathbf{x}_0)$ there exists $\mathbf{x}\in\R^n$ such that $f(\mathbf{x}) \neq 0$. In particular, if $\operatorname{supp}(f)$ is bounded, then since $\operatorname{supp}(f)$ is a closed set, it is also compact.
</MathBox>

<MathBox title='Properties of support' boxType='proposition'>
If $f, g\in\R^n \to\Gamma$ and $\gamma\in\R$ then
1. $\operatorname{supp}(f + g) \subset \operatorname{supp}(f) \cup \operatorname{supp}(f)$
2. $\operatorname{supp}(f\cdot g) \subset \operatorname{supp}(f) \cap \operatorname{supp}(g)$
3. $\operatorname{supp}(\gamma f) = \operatorname{\supp}(f)$ for $\gamma \neq 0$

<details>
<summary>Proof</summary>

**(1):** If $\mathbf{x}_0 \in \operatorname{supp}(f + g)$, then $(f + g)(\mathbf{x}) \neq 0$ for $\mathbf{x}\in\mathcal{B}_r (\mathbf{x}_0) \subset\R^n$. Consequently, $f(\mathbf{x}) \neq 0$ or $g(\mathbf{x}) \neq 0$ implying that either $\mathbf{x}_0 \in\operatorname{supp}(f)$ or $\mathbf{x}_0 \in\operatorname{supp}(g)$. Hence, $\mathbf{x}_0 \in\operatorname{supp}(f) \cup \operatorname{supp}(g)$.

**(2):** If $\mathbf{x}_0 \in\operatorname{supp}(f\cdot g)$ then $(fg)(x) \neq 0$ for $\mathbf{x}\in\mathcal{B}_r (\mathbf{x}_0)$. Consequently, $f(\mathbf{x}) \neq 0$ and $g(\mathbf{x}) \neq 0$ implying that $\mathbf{x}_0 \in\operatorname{supp}(f)$ and $\mathbf{x}_0 \in\operatorname{supp}(g)$. Hence, $\mathbf{x}_0 \in\operatorname{supp}(f) \cap \operatorname{supp}(g)$.
</details>
</MathBox>

<MathBox title='Convergence' boxType='definition'>
Consider a function $f:A\subset\R^n \to\R$. A $(f_n : A\subset\R^n \to\R)_{n\in\N}$ converges uniformly to $f$, denoted $f_n \xrightarrow{\text{u}} f$, if for every $\epsilon > 0$ there is $N(\epsilon) \in\N$ such that 

$$
  |f_n (\mathbf{x}) - f(\mathbf{x})| < \epsilon
$$

for all $\mathbf{x}\in A$ and all $n \geq N(\epsilon)$.

In the case of uniform convergence, the index $N(\epsilon)$ depends only on $\epsilon > 0$, and is the same for all $\mathbf{x}\in A$. Meanwhile, for pointwise convergence $N$ depends on $\epsilon$ and $\mathbf{x}\in A$. Thus, uniform convergence implies pointwise converge, but the converse does not necessarily hold.
</MathBox>

<MathBox title='Space of test functions' boxType='definition'>
Let $\Omega\subset\R^n$ be a compact set and consider the functions $\varphi:\R^n \to\Gamma$. The set of functions 

$$
  \mathcal{D}^m (\Omega) = \Set{\varphi|\varphi\in C^m (\R^n), \operatorname{\supp}(\varphi) \subset\Omega}
$$

is called the space of testfunction. If $\varphi\in C^m (\R^m)$ with $\operatorname{supp}(\varphi) \subset \Omega$, then $\operatorname{supp}(\D^\alpha \varphi(\mathbf{x})) \subset \operatorname{supp}(\varphi) \subset \Omega$ for $|\alpha| \leq m$. Consequently, all functions $\varphi\in C^m (\Omega)$ together with all their derivatives up to order $m$ inclusive are null outside the compact $\Omega$.

The space of test functions $\mathcal{D}^m (\Omega)$ is a vector space on $\Gamma$. The null element is the identically null function.
</MathBox>

<MathBox title='Convergence of test functions' boxType='definition'>
A sequence of test function $(\varphi_n)_{n\in \N} \subset\mathcal{D}^m (\Omega)$ converges towards $\varphi\in\mathcal{D}^m (\Omega)$, denoted $\varphi_n \xrightarrow{\mathcal{D}^m (\Omega)} \varphi$ if the sequence of functions $(\D^\alpha \varphi_n (\mathbf{x}))_{n\in\N}$ converges uniformly towards $\D^\alpha \varphi(\mathbf{x})$ in $\Omega$, hence 

$$
  \D^\alpha \varphi_n (\mathbf{x}) \xrightarrow{\text{u}} \D^\alpha \varphi(\mathbf{x}),\; 0\leq|\alpha|\leq m,\; \forall \mathbf{x}\in\Omega
$$
</MathBox>

Note that the space $\mathcal{D}^m (\Omega)$ becomes a normed vector space if we define the norm by

$$
\begin{align*}
  \lVert\varphi\rVert_{\mathcal{D}^m} =& \sup_\substack{|\alpha|\leq m \\ \mathbf{x}\in\Omega} |\D^\alpha \varphi(\mathbf{x})| \\
  =& \sup_{0\leq|\alpha|\leq m} \sup_{\mathbf{x}\in\Omega} |\D^\alpha \varphi(\mathbf{x})|, \quad \alpha\in\N^n
\end{align*}
$$

In particular, for $m = 0$, the space $\mathcal{D}^0 (\Omega)$ is denoted by $C_C^0 (\Omega)$. This is the space of continuous functions $C^0 (\R^n)$, whose support are contained in some compact set $\Omega\subset\R^n$. The test function space $C_C^0 (\Omega)$ is a normed space with norm

$$
  \varphi_{C_C^0} = \sup_{\mathbf{x}\in\Omega} |\varphi(\mathbf{x})|
$$

The sequence $(\varphi_n)_{n\in\N} \subset C_C^0 (\Omega)$ converges towards $\varphi\in C_C^0 (\Omega)$ is $\lim_{n\to\infty} \sup_{\mathbf{x}\in\Omega} |\varphi_i - \varphi| = 0$, that is, if $(\varphi_n)_{n\in\N}$ converges uniformly towards $\varphi$ in $\Omega$.
