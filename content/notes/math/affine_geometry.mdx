## Affine space

<MathBox title='Affine space' boxType='definition' tag="definition-1">
An affine space over a vector space $\vec{V}$ is any set $E$ with a map

$$
  +: E\times \vec{V} \to E
$$

with the following properties
1. **Right identity:** $\mathbf{p} + \vec{\mathbf{0}} = \mathbf{p}$ for any $\mathbf{p}\in E$
2. **Associativity:** $(\mathbf{p} + \vec{\mathbf{v}}) + \mathbf{w} = \mathbf{p} + (\mathbf{v} + \mathbf{w})$ for any $\mathbf{p}\in E$, $\mathbf{v}, \mathbf{w}\in V$
3. For every $\mathbf{a}, \mathbf{b}\in A$, there is a unique $\vec{\mathbf{v}}\in\vec{V}$ such that $\mathbf{a} + \vec{\mathbf{v}} = \mathbf{b}$. The unique vector $\vec{\mathbf{v}}$ is denoted $\vec{\mathbf{ab}}$, or alternatively $\mathbf{b} - \mathbf{a}$.

Elements of the set $E$ are called *points*. If $\mathbf{p}\in E$ and $\vec{\mathbf{v}}\in\vec{V}$, then $\mathbf{p} + \vec{\mathbf{v}}\in E$ be thought of as the point $\mathbf{p}$ translated by $\vec{\mathbf{v}}$. If $\mathbf{p},\mathbf{q}\in E$, then $\vec{\mathbf{pq}}\in\vec{V}$ can be thought of as the vector with tail at $\mathbf{p}$ and head at $\mathbf{q}$. Note that there is no distinguished point in an affine space.
</MathBox>


<MathBox title='Properties of affine space addition' boxType='proposition'>
Let $E$ be an affine space over a vector space $\vec{V}$. The for any $\mathbf{p},\mathbf{q},\mathbf{r}\in E$
1. $\vec{\mathbf{pp}} = \mathbf{0}$
2. $\vec{\mathbf{pq}} + \vec{\mathbf{qr}} = \vec{\mathbf{pr}}$
3. $\vec{\mathbf{qp}} = -\vec{\mathbf{pq}}$

<details>
<summary>Proof</summary>

**(1):** This follows immediately from the right identity property.

**(2):** From associativity
$$
\begin{align*}
  \mathbf{p} + (\vec{\mathbf{pq} + \vec{\mathbf{qr}}}) =& (\mathbf{p} + \vec{pq}) + \vec{\mathbf{qr}} \\
  =& \mathbf{q} + \vec{\mathbf{qr}}= \mathbf{r}
\end{align*}
$$

**(3):** This follows from **(1)** and **(2)** for $\mathbf{r} = \mathbf{p}$.
</details>
</MathBox>
